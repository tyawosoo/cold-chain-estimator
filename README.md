# 深海冷链时效预测工具

这是一个可直接部署到 GitHub Pages 的静态网页工具，用来根据冷链班次表快速估算“几天送到”。

## 在线访问

启用 GitHub Pages 后，访问地址将是：

- `https://tyawosoo.github.io/cold-chain-estimator/`

## 功能

- 按省 / 市 / 区县查询线路
- 选择寄出日期，自动计算下次发车时间
- 支持手动设置附加运输天数
- 自动生成可复制的分享文案
- 适配手机和电脑浏览器

## 数据说明

当前预测基于《深海冷链 冷链专线产品(运输网线图)-2022》整理出的班次表。

注意：

- 原始 Excel 是班次表，不是签收历史
- 页面中的“预计送达”属于业务估算值
- 计算逻辑是“等待下次班次天数 + 附加运输天数”

## 更新数据

如果以后 Excel 更新，可以用下面的脚本重新提取数据：

```bash
python3 generate_data.py "你的Excel文件路径.xlsx"
```

脚本位置：`generate_data.py`
